# Zip (zip)

Zip is an AI-powered procurement orchestration platform that provides a REST API for managing the full intake-to-pay lifecycle. The API enables developers to build custom integrations that extract, create, and update procurement data including intake requests, vendor onboarding, approval workflows, and contract management. Founded in 2020 and headquartered in San Francisco, Zip serves enterprise customers with a single front door for all corporate purchase requests, routing them through configurable approval chains before connecting to ERPs and financial systems. Authentication uses API keys via Basic Auth with separate sandbox and production environments. Zip also supports webhook callbacks for asynchronous event notifications when procurement workflows complete.
